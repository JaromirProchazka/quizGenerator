Detailed Description

A reversed priority queue with an item of general type and int priority.

The LOWEST priority in queue is poped the first. Implements a asymptotically Logarithmic Enque() and Pop() functions.

For example, q being the empty PriorityQueue:

char[] alphabet = new char[] { 'a', 'b', 'c' };
q.Enque(1, alphabet[1]);
q.Enque(0, alphabet[0]);
q.Enque(2, alphabet[2]);
q.Pop()

Last line returns char a, as it has priority 0
