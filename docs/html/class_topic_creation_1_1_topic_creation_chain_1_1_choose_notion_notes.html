Detailed Description

Step that lets user Choose notion page as the notes source.

This Page is downloaded of Step and is not synced with the actual notion page.

This Class uses notion page to html api For fetching the pages from URL.

Parameters
NotionPageUrlA URL link to Notion page. IMPORTANT: the page must be in the Fetch faze set as PUBLIC for acess
