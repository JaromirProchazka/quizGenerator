Detailed Description

Singleton containing methods and constants, to manage the 'quizGenerator' backend.

It follows this file structure: main_script.exe .sources\ |-------— styles.css |-------— script.js | |-------— *general topic folder\ | |-------— notes.html | |-------— questions.html | |-------— ... more topic folders\
